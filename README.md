# The Void

> *"From the darkness between worlds, something stirs."*

A new playable character mod for **Slay the Spire 2** — my first globally available mod!

![Supported Version](https://img.shields.io/badge/StS2_Version-latest-blueviolet?style=flat-square)
![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=flat-square)
![Dependency](https://img.shields.io/badge/Requires-BaseLib-orange?style=flat-square)

---

## Overview

**The Void** is a brand-new character mod that brings a fresh playstyle to the Spire.

---

## Installation

> ⚠️ **Dependency Required:** This mod requires **BaseLib** to be installed first.

1. Install **BaseLib** from the mod manager if you haven't already.
2. Download the latest release from the [Releases](https://github.com/Alchyr/BaseLib-StS2/releases) page.
3. Follow your mod manager's instructions to enable the mod.
4. Launch the game and start a new run!
