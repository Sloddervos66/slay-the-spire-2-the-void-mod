## Overview

---

## Installation

> ⚠️ **Dependency Required:** This mod requires **BaseLib** to be installed first.

1. Install **BaseLib** from the mod manager if you haven't already.
2. Download the latest release from the [Releases](https://github.com/Alchyr/BaseLib-StS2/releases) page.
3. Follow your mod manager's instructions to enable the mod.
4. Launch the game and start a new run!
